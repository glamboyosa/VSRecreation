Test Explorer Improvements

The Test Explorer now has significant performance and UI improvements. Visual Studio processes consume less memory and discovering large numbers of tests is faster. The updated UI includes easier filtering, more discoverable commands, and the addition of customizable columns and multi-tiered group by that let the user fine tune what test information is displayed

What is a Preview?

Visual Studio Preview gives you early access to the latest features and improvements not yet available in the main release. By trying Preview, you can take advantage of new builds and updates that ship on a frequent cadence. For more information on Visual Studio Preview installs, visit our FAQ page.

Please note that the preview version is continously evolving and certain features may not work as expected. This version is also not licensed to build shipping applications. If you would like to build production applications you can access stable release of Visual Studio from our downloads page.

Engage with our community

We believe that to create a great product for you we should listen to and engage with you in every stage of our development.

As you evaluate what's next with Visual Studio Preview, report a bug or suggest a feature in our developer community, where you can talk directly to our product engineers. Your feedback empowers us to evolve and ensure the highest quality in our final product.
